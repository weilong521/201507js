#网站的访问流程
1. 在浏览器中输入URL地址。
2. 把域名转成IP地址。
3. 客户端与服务器建立连接。
4. 客户端把请求发给服务器。
5. 服务器接收到请求把响应发给客户端。
6. 客户端接收响应，进行显示HTML。
7. 如果遇到外联的资源,重新向服务器发起请求这个资源,然后继续渲染.

#状态码
- 1xx 请求处理中
- 2xx 成功响应
- 3xx 重定向  302 location
- 4xx 客户端错误 404
- 5xx 服务器端错误
